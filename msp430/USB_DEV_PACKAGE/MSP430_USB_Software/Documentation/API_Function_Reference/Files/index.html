MSP430 USB Developers Package API Documentation

Introduction

This package provides API documentation for the MSP430 USB Developers Package. This package provides an easy to use API for developing USB applications on 5xx6xx MSP430 device families which have the USB peripheral.

The MSP430 USB API implements four USB device classes: the Communications Device Class (CDC), the Human Interface Device (HID) class, .the Mass Storage class (MSC) and the Personal Healthcare Device Class. It is designed for easy creation of USB devices on the MSP430.

The easiest way to get started is to click on the "Files" tab above. This will provide a list of files available in the USB Developers Package.
