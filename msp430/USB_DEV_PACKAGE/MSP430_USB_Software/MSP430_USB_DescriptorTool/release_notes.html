Installation and Usage

As part of MSPWare installation

Generally MSP USB Descriptor Tool is installed as part of MSPWare.

After installing MSPWare restart Code Composer Studio. To launch the MSP USB Descriptor Tool:

  1. Open TI Resource Explorer (View-->TI Resource Explorer). Under packages select "MSPWare".



  2. Expand Development Tools and select MSP USB Descriptor Tool.
  3. Finally, click on "Launch MSP USB Descriptor Tool".

MSP USB Descriptor Toolstandalone installation

  1. Download MSP USB Descriptor Tool installer
  2. Run the installer DescriptorTool-5_00_00_19-setup.exe for windows or DescriptorTool-5_00_00_19-Linux-x86-Install for Linux
  3. Follow installer instructions
  4. To launch tool. Go to Start Menu --> All Programs --> Texas Instruments --> MSP USB Descriptor Tool --> MSP USB Descriptor Tool_5_00_00_19
