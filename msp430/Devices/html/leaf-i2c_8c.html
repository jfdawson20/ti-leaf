Initializes Leaf I2C Bus.

This function initializes operation of the I2C Block on the Leaf board. This function must be called before any other I2C API Functions are used

Parameters
bus,Pointerto I2C Bus structure used with all Leaf I2C API Functions
mode,ConfiguresI2C Bus as Master or Slave (MODE_MASTER, MODE_SLAVE)
speed,ConfiguresI2C Bus speed (SPEED_400K, SPEED_100K)
slaveAddr,SetsSlave Address (Slave Mode Only)
Returns
0 on success, bus structure will be populated
